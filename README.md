# Diabeties-prediction
Diabetes Prediction System Using Machine Learning  . Developed a machine learning-based Diabetes Prediction System to predict the likelihood of diabetes using patient health data.
The Diabetes Prediction System is a machine learning-based healthcare application designed to predict whether an individual is likely to have diabetes based on various medical and lifestyle-related attributes. The primary objective of this project is to assist in the early detection of diabetes by analyzing patient data and providing accurate predictions, which can help healthcare professionals and individuals take preventive measures at an early stage.

The project utilizes a dataset containing important health indicators such as age, gender, body mass index (BMI), hypertension status, heart disease history, smoking history, HbA1c level, and blood glucose level. These features are significant factors associated with diabetes risk and serve as the input variables for the prediction models. The dataset undergoes several preprocessing steps, including handling categorical variables through one-hot encoding and standardizing numerical features using feature scaling techniques.

Multiple machine learning algorithms were implemented and evaluated to identify the most effective model for diabetes prediction. The models used in this project include Logistic Regression, Decision Tree, Random Forest, and Support Vector Machine (SVM). Each model was trained on the processed dataset and tested using unseen data to evaluate its predictive performance. Various evaluation metrics such as accuracy, precision, recall, F1-score, confusion matrix, and classification reports were used to compare the models comprehensively.

Among all the implemented algorithms, the Random Forest Classifier achieved the highest accuracy of approximately 97%, making it the best-performing model for this dataset. To further improve prediction reliability and leverage the strengths of multiple algorithms, an ensemble learning approach was implemented using a Soft Voting Classifier. This ensemble model combined Logistic Regression, Random Forest, and SVM predictions to produce robust and consistent results.

The project was developed using Python and popular data science libraries such as Pandas, NumPy, Scikit-learn, and Matplotlib. Overall, the Diabetes Prediction System demonstrates the practical application of machine learning in healthcare by providing a reliable and efficient tool for diabetes risk assessment, supporting data-driven decision-making and promoting early diagnosis and preventive healthcare measures.
